### Windows Release

### Linux Install
Dependencies:
- git
- python3-pip

Commands:
- `pip install pygame`
- `git clone https://git.shinysocks.net/shinysocks/raccoonundrum`
- `cd "directory where raccoonundrum was cloned"`
- `python3 main.py`
