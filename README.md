# Racoonundrum 🦝
### a trashy puzzle game created by shinysocks7
- Solve the maze to reach the trash can without getting hit by the rats which inhabit the maze.
- You are linked to the trash can so that when you move so does it.

## Movement
 - press the <kbd>w</kbd><kbd>a</kbd><kbd>s</kbd><kbd>d</kbd>  or arrow keys to move the raccoon
